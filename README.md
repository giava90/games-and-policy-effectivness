# games-and-policy-effectivness
This repo contains data and code that provides empirical evidence about the importance of communication in the policy development phase to increase its effectiveness.

The repo contains two notebooks:
- testing_H.ipynb contains the code for the DiD and creating the plot
- key_stats_and_rank_correlation_analysis.ipynb contains the code for the rank biserial correlation analysis
and two csv files (added only after publishing the paper):
- tables/scotcape_policies_clean.csv
- tables/DecisionsLandManagers.csv
